🎯 Objective

The main objective of this project is to develop a simple text-based Hangman game using Python where the player guesses a hidden word letter by letter within a limited number of attempts.

This project helps in understanding:
1.Python basics
2.Game logic building
3.User input handling
4.Conditional statements and loops

🧠 Project Description
Hangman is a word guessing game where a random word is selected from a predefined list. The player must guess the word one letter at a time. Each wrong guess reduces the number of remaining attempts. The game ends when the player either guesses the word correctly or uses all 6 attempts.

⚙️ Features
1.Random word selection from a list of 5 words
2.Maximum 6 incorrect guesses allowed
3.Real-time word display with blanks
4.Tracks already guessed letters
5.Simple console-based interface

💻 Technologies Used
Python Programming Language
Concepts: random, while loop, if-else, strings, lists

🔄 Working Process
1.A random word is selected from a predefined list.
2.The word is hidden using underscores.
3.User enters one letter per attempt.
4.If the guess is correct, letters are revealed.
5.If incorrect, attempts decrease.
6.Game continues until:
  .Word is guessed OR
  .6 wrong attempts are used

🚀 Conclusion
This project demonstrates a simple implementation of a word guessing game using Python. It is beginner-friendly and helps strengthen logical thinking and programming fundamentals.
